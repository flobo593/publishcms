Documentation of the meta.php-script

This Script is used to provide the meta information. It provides the following variables:

getauthor
Provides the Authors shown name. Specific for every single page
getpublisher
Privides the Publishers name. Usually the name of the organization the site belongs to. The same for all pages and subpages.
getdescription
Provides the site's description. Different for every single site.
getkeywords
Provides the site's keywords. Different for each site.
getpagetopic
Provides the topic of the site. The same for all pages and subpages
getpagetype
Provides the page's topig. globally the same.
getaudience
Provides the definition of the page's audience. globally the same.
getlanguage
Provides the pages langauge. Globally the same.
getrobotinfo
Provides information for the search engines crawlers
getcopyright
Provides the copyright information
gettitle
Provides the sites html-<title>
getheader
Provides the heading of the page (<h1>)
